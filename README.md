## Problem Statement
Predict whether a patient has heart disease based on medical attributes.

## Dataset
UCI Heart Disease Dataset (via Kaggle)

## Algorithms Used
- Logistic Regression
- K-Nearest Neighbors
- Decision Tree
- Random Forest
- Support Vector Machine

2. Data Cleaning

- Accuracy
- Precision
